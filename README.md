# Gerenciador de Inventário em C++

Este é um projeto simples e educativo, desenvolvido com o objetivo de praticar conceitos fundamentais da linguagem C++. O sistema permite gerenciar um inventário de produtos com funcionalidades básicas de CRUD (criação e listagem) e persistência de dados.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C++ (padrão C++11)
- **Compilador:** G++
- **Build System:** Makefile para automação
- **Controle de Versão:** Git/GitHub

## 🚀 Funcionalidades
- [x] Implementação de Classe `Produto`
- [x] Lógica de armazenamento utilizando `std::vector`
- [x] Persistência de dados em arquivo (`data/estoque.txt`)
- [x] Menu interativo para o usuário
- [x] Organização estruturada (Pastas `include/` e `src/`)
